# Introdução

:::info
Veja nosso [guia de como instalar o python em sua máquina](#), antes de iniciar a criação do bot
:::

### Requisitos
- Python 3.8 ou superior (verifique sua versão usando `python -version`)

### Instalação

Para criar nosso bot em Python, vamos usar a biblioteca [`discord.py`](https://discordpy.readthedocs.io/en/stable/), instale usando:
```sh
pip install discord.py
```
